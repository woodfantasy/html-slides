# html-slides · 下一代 AI 演示工具

> 策展级配色 × 版式骨架库 × 数据可视化原语 × 中英双语深度排版

**html-slides** (项目代号 Prismo) 是一个强大的 AI Agent Skill，专门用于生成精美的、可交互的**单文件 HTML 横向翻页演示文稿**。

无论是进行商业路演 (Pitch Deck)、行业分享、产品发布，还是技术大会演讲，html-slides 都能帮助你快速生成专业且设计感极强的在线幻灯片。

---

## 🌟 核心能力

### 1. 策展级视觉风格系统
内置 8 套经过专业设计的视觉风格，覆盖绝大多数演示场景。每套风格均包含精心调配的配色方案和排版规则，保证最终产出的专业度。
- **墨刊 · Ink Editorial**：适合人文分享、行业观察，充满杂志质感。
- **瑞士 · Swiss Grid**：适合科技产品、数据汇报，极简、严谨。
- **暗植 · Dark Botanical**：适合品牌发布、艺术展示，高端奢华。
- **信号 · Bold Signal**：适合商业 Pitch、发布会，视觉冲击力强。
- **纸墨 · Paper & Ink**：适合教育、文学叙事，独立出版物质感。
- **电压 · Creative Voltage**：适合创意提案、设计评审，充满活力。
- **素简 · Calm Minimal**：适合产品说明、冥想健康，留白与平静。
- **赛博 · Neon Cyber**：适合技术大会、开发者分享，黑客终端美学。

### 2. 25+ 注册版式骨架库
告别从零开始的排版噩梦。内置 25 种以上的版式骨架，涵盖四大类：
- **通用版式**：封面、章节过渡、金句引言、左右对比、流程步骤等。
- **数据版式**：KPI 大数字、横向/垂直时间线、矩阵网格、规格参数表等。
- **图文版式**：全幅大图、多图网格、截图美化画布等。
- **扩展版式**：代码高亮展示、特性卡片等。

### 3. 数据可视化原语
无需依赖任何外部图表库。内置纯 HTML/CSS 实现的数据展示组件，配合原生入场动效，让数据展示更生动。

### 4. 中文排版深度优化
非简单翻译西文排版规则。根据中文字符特征，进行三层字体分工、大标题字号智能分档、中英文混排优化，提供舒适的阅读体验。

### 5. 丰富的交互与动效体验
- **5 种动效 Recipe**：普通流式淡入、慢节奏仪式感、引文逐行揭示、左右滑入、按键逐步点亮。
- **WebGL 视觉背景**：流体渐变、地形等高线、粒子网格等背景，提供极佳的氛围感（在普通页面自动降低不透明度避免干扰）。
- **演讲者视图 (Speaker View)**：按下 `S` 键开启，包含双屏预览、计时器和备忘录备注。
- **低功耗模式**：按下 `B` 键一键关闭 WebGL 和动效，适应任何设备。
- **多端操控**：支持键盘方向键、空格、鼠标滚轮、触屏滑动翻页，`ESC` 键呼出缩略图索引。

### 6. 完善的配套工具链
- **PPT 导入**：提取原有 PPTX 文件的文本、图片和演讲者笔记。
- **PDF 导出**：基于 Playwright 的高保真 PDF 导出脚本，支持精简模式 (Compact Mode) 生成讲义。
- **云端部署**：一键部署到 Vercel 平台。
- **质量校验**：严密的脚本校验机制，检查版式、配色、主题节奏和多媒体规范，确保输出质量。

## 🚀 使用指南

### 启动对话
在支持此 Skill 的 AI 客户端中，你可以直接告诉 Agent 你的需求：
> "帮我制作一份关于 AI 行业趋势的分享 PPT，大概 10 页，想要专业、杂志一点的风格。"

### 典型工作流
1. **需求对齐**：Agent 会与你确认演示的受众、时长、内容大纲以及有无特殊素材。
2. **选择风格**：你可以从 8 套预设风格和配色方案中挑选最合适的一个。
3. **内容生成**：Agent 会基于你选定的风格模板，挑选合适的版式骨架填入内容。
4. **质量自检**：Agent 内部会运行 `validate.mjs` 校验产出物的质量（颜色、排版节奏等）。
5. **预览与交付**：生成完毕后，直接在浏览器中打开 `index.html` 即可开始演讲。

### 快捷键清单
在播放幻灯片时，你可以使用以下快捷键控制：
- `→` 或 `Space`：下一页 / 下一步动画
- `←`：上一页
- `ESC`：全局缩略图索引
- `S`：开启演讲者视图 (Speaker View)
- `B`：切换低功耗模式 (关闭 WebGL 和动画)

## 📁 目录结构说明

- `SKILL.md`: AI Agent 核心指令与工作流指南。
- `assets/`: 存放各风格的 HTML 种子模板及公共资源（如动效引擎）。
- `references/`: 参考指南合集（包含布局版式、组件大全、图片规范、中文排版规范等）。
- `scripts/`: 工具链脚本，用于 PPT 提取、PDF 导出、部署和质量校验。
- `examples/`: 完整的幻灯片示例。
